# contact-book-python
A simple Contact Book application built using Python. It allows users to add, search, and view contacts through a menu-driven interface.

## Features
- Add Contact
- Search Contact
- View Contacts
- Exit Program

## Technologies Used
- Python
- Dictionaries
- Loops
- Conditional Statements

## How to Run

1. Install Python
2. Run the program:

python contact_book.py
